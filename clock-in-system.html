🔥 Connect to Firebase

Toutes les données (pointages, employés, historique) seront sauvegardées en temps réel dans Firebase Firestore — accessible depuis n'importe quel appareil.

1
Va sur console.firebase.google.com → Crée un projet (ex: crm-clockin)
2
Dans ton projet → Project Settings → Add App → choisis l'icône </> Web
3
Active Firestore Database dans le menu → Build → Firestore → démarre en test mode
4
Copie ta firebaseConfig et colle les valeurs ci-dessous
